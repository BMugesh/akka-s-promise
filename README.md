# akka-s-promise

A cinematic two-page web experience built with React + Vite + TypeScript.

## Project Structure

- **`/`** → StockGenie (fake project hero page — the disguise)
- **`/akka`** → The real emotional website

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## Tech Stack

- React 18 + TypeScript
- Vite
- Framer Motion (animations & parallax)
- Tailwind CSS
- React Router v6
